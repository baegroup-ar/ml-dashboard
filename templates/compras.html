Costos de mercadería (CMV)

Cargá el costo de cada SKU sin IVA, con la fecha de vigencia y la tasa de IVA que le corresponde (21%, 10.5%, 0 si está exento, etc.). Cada venta toma el costo vigente al momento de la venta y el sistema aplica la tasa automáticamente para que sea comparable con monto/comisión/envío de ML, que vienen con IVA incluido. Los costos se comparten entre todas las cuentas de la empresa: no hace falta elegir cuenta.

Cargar desde Excel o CSV

⬇ Descargar cargados Descargar plantilla
Columnas esperadas: SKU, Costo (sin IVA), opcionalmente Fecha (vigencia desde, YYYY-MM-DD o DD/MM/YYYY; si falta, hoy), IVA (tasa: 21, 10.5, 0 o Exento; default 21%), Cantidad (opcional: si la cargás, esa fila cuenta como una compra real en Métricas) y Proveedor (opcional; si la dejás vacía no borra el proveedor ya cargado). Cada combinación SKU+Fecha es una versión histórica. Bajate la plantilla para tener un Excel listo con las columnas y filas de ejemplo.

Agregar / actualizar manualmente

Si dejás la fecha vacía toma hoy. Tasa por defecto 21%, escribí 10.5 o 0 según corresponda. Cantidad es opcional: cargala solo si esto es una compra real que querés ver en Métricas.
